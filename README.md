A modern, user-friendly Quality Point Average (QPA) calculator built with Next.js and Tailwind CSS. This tool helps Carnegie Mellon University students calculate their QPA by managing semesters and courses.

## Features

- Import courses directly from CMU academic record PDFs
- Add multiple semesters manually
- Add courses with names, grades, and units
- Real-time QPA calculation
- Data stored locally in your browser - no server uploads
- Clean and responsive interface
- Built with modern web technologies

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Shadcn UI Components
- PDF.js for academic record parsing

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/miguel-salv/QPA-Calculator.git
   cd QPA-Calculator
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

## Running the Project

Run the development server:

```bash
npm run dev
```

Open [http://localhost:9002](http://localhost:9002) in your browser

## Dependencies

- **Next.js:** React framework for building web applications.
- **TypeScript:** Superset of JavaScript that adds static typing.
- **Tailwind CSS:** Utility-first CSS framework for styling.
- **Shadcn UI Components:** Reusable UI components built with Radix UI and Tailwind CSS.
- **PDF.js:** JavaScript library to render PDFs.
- **lucide-react:** Set of icons for React.
- **react-hook-form**: Library for form management and validation.
- **zod**: Library for schema declaration and validation.
